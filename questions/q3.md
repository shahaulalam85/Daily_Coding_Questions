# Equilibrium Point

**Platform:** GeeksforGeeks  
**Topic:** Array / Prefix Sum  
**Difficulty:** Easy  

---

## 🧠 Idea in One Line
Use total sum and maintain left sum; compute right sum dynamically.

---

## 🔍 Key Observation
Right sum = total sum − left sum − current element

---

## 🚀 Approach
- Calculate total sum of array
- Traverse array
- Maintain left sum
- Compare left sum and right sum

---

## 🪜 Algorithm Steps
1. Calculate total sum of array
2. Initialize `left = 0`
3. Loop through array
4. Compute `right = total − left − arr[i]`
5. If `left == right`, return index
6. Update `left += arr[i]`

---

## ⏱ Time Complexity
O(n)

## 📦 Space Complexity
O(1)

---

## ⚠️ Edge Cases
- Single element array
- All elements same
- Negative numbers
- No equilibrium index

---

## 💻 Code Pattern to Remember
```cpp
class Solution {
public:
    int findEquilibrium(vector<int> &arr) {
        long long total = 0, left = 0;

        for(int x : arr) total += x;

        for(int i = 0; i < arr.size(); i++){
            long long right = total - left - arr[i];
            if(left == right) return i;
            left += arr[i];
        }

        return -1;
    }
};
