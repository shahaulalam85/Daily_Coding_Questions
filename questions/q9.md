# A. Game with Integers

**Difficulty:** Easy  

---

## 🧠 Idea in One Line
Check if Vanya can make the number divisible by 3 in one move.

---

## 🔍 Key Observation
- Vanya can add or subtract 1
- He wins if result divisible by 3
- So if `(n-1)%3 == 0` OR `(n+1)%3 == 0` → First wins
- Otherwise Second wins

---

## 🚀 Approach
- For each n
- Check divisibility condition
- Print winner

---

## 🪜 Algorithm Steps
1. Read test cases
2. For each test read `n`
3. Check `(n-1)%3 == 0`
4. OR check `(n+1)%3 == 0`
5. If true print "First"
6. Else print "Second"

---

## ⏱ Time Complexity
O(t)

## 📦 Space Complexity
O(1)

---

## ⚠️ Edge Cases
- n = 1
- n already divisible by 3
- large n
- n = 2
- n = multiple of 3

---

## 💻 Code Pattern to Remember
```cpp
#include <bits/stdc++.h>
using namespace std;

int main(){
    int t;
    cin >> t;

    while(t--){
        int n;
        cin >> n;

        if((n-1)%3==0 || (n+1)%3==0)
            cout << "First" << endl;
        else
            cout << "Second" << endl;
    }

    return 0;
}
