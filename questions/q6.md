# A. Friendly Numbers

## 🖼 Problem 6
![Friendly Numbers](/Images/q6.png)

---

**Platform:** Codeforces  
**Topic:** Math / Brute Force  
**Difficulty:** Easy  

---

## 🧠 Idea in One Line
Check numbers near x and count those satisfying `y - digitSum(y) = x`.

---

## 🔍 Key Observation
- Maximum digit sum for numbers ≤ 1e9 is 9 * 10 = 90  
- So valid y must lie in range `[x , x + 90]`  
- Only check limited range

---

## 🚀 Approach
- Iterate from `x` to `x + 90`
- Compute digit sum
- Check condition
- Count valid numbers

---

## 🪜 Algorithm Steps
1. Read test cases
2. For each test read `x`
3. Initialize count = 0
4. Loop from `i = x` to `x + 90`
5. Compute digit sum
6. If `i - digitSum(i) == x` increment count
7. Print count

---

## ⏱ Time Complexity
O(90 * t) ≈ O(t)

## 📦 Space Complexity
O(1)

---

## ⚠️ Edge Cases
- x = 0
- large x (1e9)
- no friendly numbers
- multiple friendly numbers
- x very small

---

## 💻 Code Pattern to Remember
```cpp
#include <iostream>
using namespace std;

long long digitSum(long long n){
    long long s =0;
    while(n>0){
        s += n%10;
        n /= 10;
    }
    return s;
}

int main(){
    int t;
    cin >> t;
    while(t--){
        long long x;
        cin >> x;

        int count =0;

        for(long long i=x; i<=x+90; i++ ){
            if(i-digitSum(i)==x){
                count ++;
            }
        }

        cout << count << endl;
    }
    return 0;
}
