# 1480. Running Sum of 1d Array

## 🖼 Problem Screenshot
![Running Sum](/Images/q14.png)

---

**Platform:** LeetCode  
**Topic:** Prefix Sum / Array  
**Difficulty:** Easy  

---

## 🧠 Idea in One Line
Each element becomes sum of itself and all previous elements.

---

## 🔍 Key Observation
Running sum: running[i] = running[i-1] + nums[i]

---

## 🚀 Approach
- Create result array
- Copy first element
- Add previous sum iteratively

---

## 🪜 Algorithm Steps
1. Read array
2. Create result array
3. Set first element
4. Loop from index 1
5. Add previous running sum
6. Return result

---

## ⏱ Time Complexity
O(n)

## 📦 Space Complexity
O(n)

---

## ⚠️ Edge Cases
- single element array
- negative numbers
- all zeros
- large values
- empty array (not in constraints)

---

## 💻 Code Pattern to Remember
```cpp
class Solution {
public:
    vector<int> runningSum(vector<int>& nums) {
        int n = nums.size();
        vector<int> a(n);

        a[0] = nums[0];

        for(int i = 1; i < n; i++){
            a[i] = a[i-1] + nums[i];
        }

        return a;
    }
};
