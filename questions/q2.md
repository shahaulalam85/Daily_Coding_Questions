## 🧠 Idea in One Line
Track minimum price so far and calculate maximum profit.

---

## 🔍 Key Observation
Profit = current price − minimum price seen before

---

## 🚀 Approach
- Maintain minimum price
- Calculate profit at each step
- Track maximum profit

---

## 🪜 Algorithm Steps
1. Initialize `minPrice = INT_MAX`
2. Initialize `profit = 0`
3. Traverse array
4. Update minPrice
5. Calculate profit = price - minPrice
6. Update max profit

---

## ⏱ Time Complexity
O(n)

## 📦 Space Complexity
O(1)

---

## ⚠️ Edge Cases
- All decreasing prices
- Single element
- Same prices
- Profit = 0 case

---

## 💻 Code Pattern to Remember
```cpp
class Solution {
public:
    int maxProfit(vector<int>& prices) {
        int mn = INT_MAX;
        int profit = 0;

        for(int price : prices){
            mn = min(mn, price);
            profit = max(profit, price - mn);
        }

        return profit;
    }
};
