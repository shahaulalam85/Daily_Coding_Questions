# A. Was there an Array?

## 🖼 Problem 20
![Was there an Array](/Images/q20.png)

---

**Platform:** Codeforces  
**Topic:** Greedy / Pattern Checking  
**Difficulty:** Easy  

---

## 🧠 Idea in One Line
Array is impossible if pattern `1 0 1` appears in the given sequence.

---

## 🔍 Key Observation
- If `b[i] = 1`, then three consecutive elements in original array must be equal
- Pattern `1 0 1` creates contradiction
- So check if sequence contains `1 0 1`

---

## 🚀 Approach
- Read array
- Check for pattern `1 0 1`
- If found → NO
- Otherwise → YES

---

## 🪜 Algorithm Steps
1. Read test cases
2. For each test read `n`
3. Read array of size `n-2`
4. Loop from index 1 to n-3
5. Check if `arr[i-1]==1 && arr[i]==0 && arr[i+1]==1`
6. If found print NO
7. Else print YES

---

## ⏱ Time Complexity
O(n)

## 📦 Space Complexity
O(n)

---

## ⚠️ Edge Cases
- n = 3 (smallest size)
- all zeros
- all ones
- single pattern
- multiple patterns

---

## 💻 Code Pattern to Remember
```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int t;
    cin >> t;

    while (t--)
    {
        int n;
        cin >> n;

        int arr[n - 2];
        for (int i = 0; i < n - 2; i++)
            cin >> arr[i];

        int flag = 0;

        for (int i = 1; i < n - 3; i++)
        {
            if (arr[i - 1] == 1 && arr[i] == 0 && arr[i + 1] == 1)
            {
                flag = 1;
                break;
            }
        }

        if (flag)
            cout << "NO" << endl;
        else
            cout << "YES" << endl;
    }

    return 0;
}
