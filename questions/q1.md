---

## 🧠 Idea in One Line
Traverse from right and keep track of maximum seen so far.

---

## 🔍 Key Observation
Rightmost element is always a leader.

If current element ≥ max_so_far → it is a leader.

---

## 🚀 Approach
- Start from rightmost element
- Maintain `max`
- If element ≥ max → add to answer
- Update max
- Reverse result

---

## 🪜 Algorithm Steps
1. Initialize max = last element
2. Add last element to answer
3. Traverse from right to left
4. If arr[i] ≥ max
5. Update max = arr[i]
6. Push to answer
7. Reverse answer

---

## ⏱ Time Complexity
O(n)

## 📦 Space Complexity
O(n)

---

## ⚠️ Edge Cases
- Single element
- All increasing
- All decreasing
- Duplicate elements

---

## 💻 Code Pattern to Remember
```cpp
class Solution {
public:
    vector<int> leaders(vector<int>& arr) {
        int n = arr.size();
        vector<int> ans;

        int max = arr[n-1];
        ans.push_back(max);

        for(int i=n-2; i>=0; i--){
            if(arr[i] >= max){
                max = arr[i];
                ans.push_back(max);
            }
        }

        reverse(ans.begin(), ans.end());
        return ans;
    }
};
