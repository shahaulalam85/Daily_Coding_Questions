# A. Turtle and Good Strings

**Platform:** Codeforces  
**Topic:** Greedy / String  
**Difficulty:** Easy  

---

## 🧠 Idea in One Line
String is good if first and last characters are different.

---

## 🔍 Key Observation
- We can split string into two parts
- Condition satisfied if first char ≠ last char
- Otherwise impossible

---

## 🚀 Approach
- Compare first and last character
- If different → YES
- Else → NO

---

## 🪜 Algorithm Steps
1. Read test cases
2. For each test read `n`
3. Read string `s`
4. Compare `s[0]` and `s[n-1]`
5. If not equal print YES
6. Else print NO

---

## ⏱ Time Complexity
O(1) per test

## 📦 Space Complexity
O(1)

---

## ⚠️ Edge Cases
- n = 2
- all same characters
- alternating characters
- first and last same
- first and last different

---

## 💻 Code Pattern to Remember
```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int t;
    cin >> t;

    while (t--)
    {
        int n;
        cin >> n;

        string s;
        cin >> s;

        if (s[0] != s[n - 1])
            cout << "YES" << endl;
        else
            cout << "NO" << endl;
    }

    return 0;
}
