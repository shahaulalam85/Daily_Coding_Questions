# A. Line Trip

## 🖼 Problem 7
![Line Trip](/Images/q7.png)

---

**Platform:** Codeforces  
**Topic:** Greedy / Math  
**Difficulty:** Easy  

---

## 🧠 Idea in One Line
Answer is the maximum gap between consecutive stops, considering return trip needs double fuel.

---

## 🔍 Key Observation
- Need fuel for largest distance without refueling
- Forward gaps → `a[i] - a[i-1]`
- First gap → `a[0] - 0`
- Return trip from x → `2 * (x - last_station)`

---

## 🚀 Approach
- Track maximum gap between stations
- Also check return distance from last station
- Take maximum of all

---

## 🪜 Algorithm Steps
1. Read test cases
2. For each test case read `n , x`
3. Read array of stations
4. Initialize `ans = a[0]`
5. Compute max gap between consecutive stations
6. Compute `2*(x - last_station)`
7. Print maximum

---

## ⏱ Time Complexity
O(n) per test case

## 📦 Space Complexity
O(n)

---

## ⚠️ Edge Cases
- Only one gas station
- station very close to x
- station very close to 0
- large last gap
- n = 1

---

## 💻 Code Pattern to Remember
```cpp
#include <iostream>
#include <algorithm>
#include <vector>
using namespace std;

int main()
{
    int t;
    cin >> t;
    while (t--)
    {
        int n, x;
        cin >> n >> x;

        vector<int> a(n);
        for (int i = 0; i < n; i++)
            cin >> a[i];

        int ans = a[0];

        for (int i = 1; i < n; i++)
            ans = max(ans, a[i] - a[i - 1]);

        ans = max(ans, 2 * (x - a[n - 1]));

        cout << ans << endl;
    }
}
