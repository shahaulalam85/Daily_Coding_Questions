# A. Halloumi Boxes

## 🖼 Problem 5
![Halloumi Boxes](/Images/q5.png)

---

**Platform:** Codeforces  
**Topic:** Greedy / Sorting  
**Difficulty:** Easy  

---

## 🧠 Idea in One Line
If k > 1, array can always be sorted; if k == 1, array must already be sorted.

---

## 🔍 Key Observation
- When k = 1 → only subarray of length 1 → no change possible  
- When k ≥ 2 → any permutation achievable using reverses  
- So only check sorted condition when k == 1

---

## 🚀 Approach
- Read array
- If k == 1 check sorted
- Else always YES

---

## 🪜 Algorithm Steps
1. Read test cases
2. For each test case read `n , k`
3. Read array
4. If `k == 1` check `is_sorted`
5. If sorted print YES else NO
6. If `k > 1` print YES

---

## ⏱ Time Complexity
O(n) per test case

## 📦 Space Complexity
O(n)

---

## ⚠️ Edge Cases
- k = 1 and array unsorted
- already sorted array
- n = 1
- duplicate elements
- k = n

---

## 💻 Code Pattern to Remember 
```cpp
#include <iostream>
#include <algorithm>
#include <vector>
using namespace std;

int main()
{
    int t;
    cin >> t;
    while (t--)
    {
        int n, k;
        cin >> n >> k;

        vector<int> a(n);
        for (int i = 0; i < n; i++)
            cin >> a[i];

        if (k == 1)
        {
            if (is_sorted(a.begin(), a.end()))
                cout << "YES\n";
            else
                cout << "NO\n";
        }
        else
            cout << "YES\n";
    }

    return 0;
}
