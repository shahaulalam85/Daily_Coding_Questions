# A. Cover in Water

## 🖼 Problem 8

**Platform:** Codeforces  
**Topic:** Greedy / String  
**Difficulty:** Easy  

---

## 🧠 Idea in One Line
If three consecutive empty cells exist answer is 2, otherwise count empty cells.

---

## 🔍 Key Observation
- Pattern "..." allows filling all with just 2 operations
- Otherwise each '.' needs separate operation
- So check if substring "..." exists

---

## 🚀 Approach
- Count empty cells
- Check if "..." exists
- If yes → answer = 2
- Else → answer = number of empty cells

---

## 🪜 Algorithm Steps
1. Read test cases
2. For each test read `n` and string `s`
3. Initialize `empty = 0`
4. Check if substring "..." exists
5. Count '.' characters
6. If found print 2
7. Else print empty

---

## ⏱ Time Complexity
O(n)

## 📦 Space Complexity
O(1)

---

## ⚠️ Edge Cases
- no empty cells
- only one empty cell
- exactly three empty cells
- empty cells at boundary
- n = 1

---

## 💻 Code Pattern to Remember
```cpp
#include <bits/stdc++.h>
using namespace std;

int main(){
    int t;
    cin >> t;

    while(t--){
        int n;
        string s;
        cin >> n >> s;

        bool empty_three = false;
        int empty = 0;

        for(int i=0; i<n; i++){
            if(i+2<n && s[i]=='.' && s[i+1]=='.' && s[i+2]=='.'){
                empty_three = true;
                break;
            }
            if(s[i]=='.') empty++;
        }

        if(empty_three) cout << 2 << endl;
        else cout << empty << endl;
    }

    return 0;
}
