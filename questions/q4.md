# A. Parkour Design

## 🖼 Problem 4

**Platform:** Codeforces  
**Topic:** Math / Greedy  
**Difficulty:** Easy  

---

## 🧠 Idea in One Line
Convert moves into equations and check divisibility condition.

---

## 🔍 Key Observation
Allowed moves:
- (2, +1) => 'a' times
- (3, 0) => 'b' times
- (4, -1) => 'c' times
- a , b, c are +ve

After forming equations:
x - 2*y must be divisible by 3

---

## 🚀 Approach
- Convert moves into linear equations
- Derive condition
- Check divisibility
- Ensure valid values
- x = 2a + 3b + 4c
- y = a - c

---

## 🪜 Algorithm Steps
1. Read number of test cases
2. For each test case read `x , y`
3. Compute `diff = x - 2*y`
4. Check if `diff % 3 == 0`
5. Check if `diff >= 0`
6. Print YES else NO

---

## ⏱ Time Complexity
O(t)

## 📦 Space Complexity
O(1)

---

## ⚠️ Edge Cases
- Negative y values
- diff becomes negative
- diff not divisible by 3
- Large x values (1e9)
- x = 0

---

## 💻 Code Pattern to Remember
```cpp
#include <iostream>
using namespace std;

int main(){
    int t;
    cin >> t;
    while(t--){
        long long x, y;
        cin >> x >> y;

        long long diff = x - 2*y;

        if(diff % 3 == 0 && diff >= 0 && x + 4*y >= 0)
            cout << "YES\n";
        else
            cout << "NO\n";
    }
    return 0;
}
